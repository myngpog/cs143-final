### Overview
This is a basic, working version of twitter. It includes the use of Python, Flask, HTML/CSS, Jinja2, and SQLite3. 

### Build Instructions
To build:
```
# Build
docker-compose up -d --build

# Take down
docker-compose down

# Take down volumes as well 
docker-compose down -v
```
